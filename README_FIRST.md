&nbsp;         ANLEITUNG FÜR LINUX (start.sh)

=====================================================



Hallo Papa, damit der Bot unter Linux läuft, sind zwei 

kleine Schritte im Terminal nötig:



1\. BERECHTIGUNG FREISCHALTEN:

&nbsp;  Linux erlaubt das Ausführen von Skripten erst nach 

&nbsp;  einer expliziten Freigabe. Öffne das Terminal in 

&nbsp;  diesem Ordner und tippe:



&nbsp;  chmod +x Start\_MC Stats Bot.sh



2\. BOT STARTEN:

&nbsp;  Tippe danach einfach diesen Befehl ein, um den Bot 

&nbsp;  zu starten (er installiert dann alles Nötige selbst):



&nbsp;  ./Start\_MC Stats Bot.sh



HINWEIS:

Sollte "node" nicht gefunden werden, muss Node.js 

installiert sein. Meistens geht das unter Linux mit:

sudo apt install nodejs npm
